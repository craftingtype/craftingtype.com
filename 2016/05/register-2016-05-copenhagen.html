Font Editing Software

Attendees are welcome to use any font editing application they wish. At this event, we can best support FontForge, FontLab Studio, Glyphs, Fontographer, and TypeTool.

FontForge

Crafting Type is happy to welcome FontForge as a sponsor. FontForge offers:

  • access to a fully-functional version of FontForge forever, for free
  • complete access to the development process on Github

You can directly improve FontForge by giving the project developers a tip. We work with freelance programmer Frank Trampe to fix issues that matter most to workshop participants. If you are in the USA, donations to TUG are fully tax deductible.

If you choose to pay via PayPal, the results page will contain a button to make the payment. Our paypal address is paypal@tug.org.

If you wish to pay by check or bank transfer, please print this page and mail it to "TeX Users Group - font workshops, PO Box 2311, Portland, OR 97208-2311, USA" or fax it to +1 815-301-3568, and email office-admin@tug.org to say you are doing this.
