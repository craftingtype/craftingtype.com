VirtualBox — Windows, Mac OS X, GNU+Linux

  1. Grab a USB key from Dave
  2. Copy these files to your computer
    • CT.vdi
    • VirtualBox-4.2.4-81684-OSX.dmg if you have a Mac
    • VirtualBox-4.2.4-81684-Win.exe if you have Windows
    • VirtualBox Extension Pack
  3. Install VirtualBox in the normal way
  4. Double click the Extension Pack file to install it
    • Scroll to the bottom, click Agree
  5. Reboot your computer so it is fresh
  6. Run VirtualBox
  7. Go to the File menu, Preferences/Settings option
  8. Go to Input
    • Set the host key to Right Alt
    • Tick Autocapture
    • click OK
  9. Go to Machine menu, click New
    • Name: Crafting Type
    • OS: Linux
    • Version: Red Hat
  10. Click Next
  11. Set Memory to the highest amount of green
    • This should be 1,020 Mb or more
  12. Click Next
  13. Tick Startup disk and Use existing hard disk
  14. Click the folder icon, and locate the CT.vdi file
  15. You’ll see it reads the file and says its 18Gb — the file will balloon in size as you add stuff to it, up to a maximum limit of 18Gb
  16. Click Next
  17. Click Create
  18. In the list on the left, select Crafting Type, on the toolbar click Settings
    • General, Advanced
      • Set Shared Clipboard to Bidirectional
    • Display
      • Crank the video memory up to 80% (eg, around 100Mb if the total is 128Mb)
    • Shared Folders:
      • Click the little folder with a + icon
      • Click Folder Path’s little drop down
      • Click Other
      • Find a folder you want to share between the VirtualBox computer and your real one (perhaps your Desktop folder) but don’t enter it, just click it once and then hit the choose button
      • Set Folder Name to Shared
      • Tick Auto-Mount
      • Click OK
  19. Now on the toolbar click Start
    • It will warn you that some nice features aren’t working, and you can tick the box to not tell you this again, because when the virtual computer boots up then all these things will start working
    • The first boot can be a little slow as it settles down in its new home.
    • Wait patiently until you see a colorful blue desktop
  20. Go to the View menu, select Full Screen
    • You’ll get another helpful explanation about how your real computer is running in the background and its easy to exit full screen mode - use the appearing toolbar at the bottom of the screen, or hold the ‘host key’ you set earlier (Right Alt) and tap f
